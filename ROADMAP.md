# LinkPunch — Roadmap

## ✅ Completed
- 3D link-in-bio with Three.js / React Three Fiber
- Glass-morphic banner cards with flip interaction
- Creator analytics (page views, click tracking, CTR)
- WebGL fallback (plain link list)
- Mobile-responsive banner sizing
- Accessibility: prefers-reduced-motion, focus-visible
- Skeleton loading states during 3D initialization

- [ ] Safe-area insets for notched devices (iPhone, Android)
- [ ] Touch-target sizing (44px minimum) for mobile
- [ ] Shimmer skeleton placeholders during load
